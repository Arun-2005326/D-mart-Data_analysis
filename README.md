# D-Mart Data Analysis Dashboard 📊

## Project Overview

This project presents an interactive sales analysis dashboard created using Microsoft Excel. The dashboard provides valuable insights into customer demographics, sales performance, order status, sales channels, and regional trends.

The objective of this project is to analyze sales data and identify business patterns that can help improve decision-making and customer targeting.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning and Transformation

## Dataset Information

The dataset contains customer orders, product categories, sales amounts, order status, channels, age groups, gender, and location details.

### Key Fields

* Order ID
* Customer ID
* Gender
* Age
* Category
* Sales Amount
* Order Status
* Sales Channel
* State
* City

## Dashboard Features

### Sales vs Orders Analysis

* Monthly comparison of total sales and order volume.

### Order Status Analysis

* Delivered Orders
* Cancelled Orders
* Returned Orders
* Refunded Orders

### Top 5 States by Sales

* Karnataka
* Maharashtra
* Tamil Nadu
* Telangana
* Uttar Pradesh

### Sales Channel Performance

* Amazon
* Flipkart
* Myntra
* Ajio
* Meesho
* Others

### Customer Demographics

* Men vs Women Sales Analysis
* Age Group vs Gender Analysis

### Interactive Filters

* Month Filter
* Product Category Filter
* Sales Channel Filter

## Key Insights

* Women contributed approximately 64% of total sales.
* Adult customers generated the highest sales volume.
* Amazon was the leading sales channel.
* Most orders were successfully delivered.
* Karnataka and Maharashtra were among the highest revenue-generating states.
